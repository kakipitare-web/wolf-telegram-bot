=====================================
שלב 1: התקנת Node.js (רק פעם אחת)
=====================================
1. כנס ל: https://nodejs.org/
2. הורד את הגרסה LTS
3. התקן

בדיקה: פתח CMD והקלד:
    node -v

=====================================
שלב 2: הורדת התבנית מ-Canva
=====================================
1. פתח את העיצוב ב-Canva:
   https://www.canva.com/d/oQ5x7kfYaHMiiAG

2. לחץ "Share" → "Download"
3. בחר:
   - File type: PNG
   - Size: 1080 x 1920
4. שמור את הקובץ בשם: template.png
5. העבר אותו לתיקייה: C:\Users\danie\Downloads\wolf-telegram-bot

=====================================
שלב 3: הגדרת ה-TOKEN
=====================================
1. העתק את הקובץ .env.example
2. קרא לעותק בשם: .env (בלי סיומת)
3. פתח אותו בפנקס רשימות
4. החלף את "your_token_here" ב-TOKEN שלך מה-BotFather
5. שמור

⚠️ הערת אבטחה: ה-TOKEN ששלחת בצ'אט חשוף — אחרי שהבוט עובד,
   עבור ל-BotFather ושלח /revoke כדי לקבל TOKEN חדש

=====================================
שלב 4: התקנת החבילות (רק פעם אחת)
=====================================
1. פתח CMD
2. הקלד:
    cd C:\Users\danie\Downloads\wolf-telegram-bot
    npm install

(זה לוקח 2-3 דקות)

=====================================
שלב 5: הרצה
=====================================
בCMD, באותה תיקייה:
    npm start

אתה אמור לראות:
    🐺 Wolf Bot started. Waiting for commands...

=====================================
שימוש בבוט
=====================================

פקודות:
/start - תפריט עזרה
/win   - ייצור פוסט "נתפס" (עם תוצאה)
/tip   - ייצור פוסט לטיפ חדש (בלי תוצאה)

דוגמה לשימוש:
    /win
    match: Barcelona vs Real Madrid
    pick: Over 2.5 Goals
    odds: 1.85
    score: 3-1

עם צילום מסך:
    1. שלח את הצילום מסך מפלאשסקור לבוט
    2. עשה עליו Reply ושלח:
       /win
       match: ...
       pick: ...
       odds: ...
       score: ...

=====================================
התאמת מיקום טקסטים
=====================================
אם המיקומים לא מושלמים על התבנית שלך,
פתח את bot.js וחפש את הבלוק LAYOUT.

כל פריט (match, pick, odds, score) יש לו:
    x: מיקום אופקי (0 = שמאל, 1080 = ימין)
    y: מיקום אנכי (0 = למעלה, 1920 = למטה)
    font: גודל וסוג הפונט
    color: צבע

תשנה את המספרים ותריץ מחדש (Ctrl+C ואז npm start).
